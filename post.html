Online Ordering System

This was a project We started back in June 2019 when We do our thesis. I was the lead programmer of this project both front end and back end.

The system stored customer orders and their details. Customer had the ability to view all their data such as orders and their details.

Admin had the ability to manage customer orders, maintain system and generate reports.

System generated different reports managed by admin.

We worked on this system for almost one semester included the documentation.

Technologies:
  • - HTML
  • - CSS
  • - Javascript
  • - Bootstrap
  • - PHP
  • - MySql
  • - Photoshop
